Import Consumption from Order History

Upload the Order History file (SKU Code SKU Qty Request Date). Daily rates use only the last 60 days of orders by default (rolling window, anchored to the latest order date) — older consumption is ignored, and SKUs with no orders in the window are reset to 0.

⚙ Automated: if the order history lives in an Orders tab of the connected Google Sheet, rates can be recalculated without any upload (and auto-refresh daily via the Apps Script trigger).

📊 Rates derived from historical orders data · used to compute DOI = stock ÷ daily rate
